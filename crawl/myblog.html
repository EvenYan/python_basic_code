Blog项目(一):环境搭建及模型设计

Blog项目 2017年1月11日 09:51 even 4 评论 384 阅读

开发环境

软件及版本

  • Python: Python 3.6
  • 虚拟环境：virtualenv，virtualenvwrapper
  • IDE: PyCharm
  • Django: Django 1.11.7
  • 系统: Ubuntu 16.04
  • 数据库: MySQL + Redis
  • 虚拟环境: pipenv

为了避免麻烦，建议你的环境尽可能与本文一致。Python 版本最好是3.4 或以上，Django版本号必须为 最好是1.11(LTS:长期支持版)。

项目需求

创建项目

# 创建虚拟环境
mkvirtalenv blog

# 进入虚拟环境
workon blog

# 安装django
pip install django==1.11.7

# 创建项目
django-admin startproject myblog

# 运行项目
cd myblog
python manage.py runserver

# 在myblog/myblog/settings.py文件中修改语言和时区
LANGUAGE_CODE = 'zh-hans'
TIME_ZONE = 'Asia/Shanghai'

# 创建app
python manage.py startapp blog

# 在myblog/myblog/settings.py文件中将app注册到项目
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    ......
    'blog', # 注册 blog 应用
]

设计数据库模型

文章表

  • 文章ID

  • 标签：eg：python, django, blog

分类表

标签表

  • 标签ID
  • 标签名称
# myblog/models.py

from django.db import models
from django.contrib.auth.models import User


class Category(models.Model):
    name = models.CharField(max_length=100)


class Tag(models.Model):
    name = models.CharField(max_length=100)


class Post(models.Model):
    title = models.CharField(max_length=70)
    body = models.TextField()
    created_time = models.DateTimeField()
    modified_time = models.DateTimeField()
    excerpt = models.CharField(max_length=200, blank=True)
    category = models.ForeignKey(Category)
    tags = models.ManyToManyField(Tag, blank=True)
    author = models.ForeignKey(User)
